## 📚 Book Scraper ETL Pipeline

ETL-пайплайн для сбора, очистки и сохранения данных о книгах. Проект создан в рамках обучающего курса по Data Engineering.

### 📝 Описание проекта

Этот скрипт собирает данные из книжного онлайн-каталога (books.toscrape.com), имитируя поведение реального пользователя. Затем он очищает «сырые» данные с помощью Pandas (нормализация цен, конвертация текстовых рейтингов в числа) и загружает лучшие книги (рейтинг 4 и 5) в реляционную базу данных PostgreSQL.

### Ключевые особенности:
* **Умный парсинг:** Обход защиты от ботов (User-Agent), обработка ошибок соединения (Retries) и вежливые паузы между запросами (Timeouts).
* **Автоматическая пагинация:** Скрипт сам находит кнопку "Next" и идет по каталогу до самой последней страницы.
* **Логирование:** Использование библиотеки `logging` вместо обычных принтов для отслеживания каждого этапа (INFO, WARNING, ERROR).
* **Data Quality:** Проверка данных на пустоты (NaN) и приведение типов перед загрузкой в БД.

---

### 🛠️ Технологический стек

* **Python 3.x** — основной язык логики.
* **Requests & BeautifulSoup4** — инструменты для извлечения данных (Extract).
* **Pandas** — очистка, агрегация и трансформация данных (Transform).
* **SQLAlchemy & PostgreSQL** — загрузка и хранение данных (Load).

---

### ⚙️ Структура проекта

* `scraper_etl.py` — основной скрипт, выполняющий полный цикл ETL.
* `read_db.py` — тестовый скрипт для чтения и проверки данных из PostgreSQL.
* `books_cleaned.csv` — локальная резервная копия очищенных данных (создается автоматически).

---

### 🚀 Как запустить проект

#### 1. Подготовка окружения
Клонируйте репозиторий и установите необходимые библиотеки:
```bash
pip install requests beautifulsoup4 pandas sqlalchemy psycopg2-binary
```

2. Настройка базы данных
Убедитесь, что у вас установлен и запущен PostgreSQL. Создайте базу данных (например, my_library).
Откройте файл scraper_etl.py и обновите строку подключения DB_URL своими данными:

```Python
DB_URL = "postgresql://ваш_пользователь:ваш_пароль@localhost:5432/my_library"
```

3. Запуск пайплайна
Запустите основной скрипт:

```Bash
python scraper_etl.py
```
Вы увидите подробный лог работы в консоли. После завершения парсинга отфильтрованные данные будут лежать в вашей таблице top_books.

4. Проверка результата
Запустите проверочный скрипт, чтобы прочитать данные прямо из базы:

```Bash
python read_db.py
```
